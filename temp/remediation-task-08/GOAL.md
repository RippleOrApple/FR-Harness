# Goal

## Objective

使用两个全新 OpenCode 会话做规格合规和代码质量评审，验证意见后修复，并创建正式 GitHub PR。

## Scope

- 评审前全量测试、演示和凭据扫描。
- OpenCode 规格合规 review。
- OpenCode 安全/质量 review。
- 人工逐项验证 Critical/Major。
- 推送 `setup-scaffold`、创建 PR、发布两份评审摘要。

## Acceptance Criteria

- 两份 review 有原始结论和人工判断。
- 所有确认的 Critical/Major 已修复并测试。
- 分支推送，PR URL 可访问。
- 历史偏差没有被倒签。
